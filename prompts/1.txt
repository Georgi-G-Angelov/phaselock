You are implementing Step 1 of the PhaseLock application — a cross-platform desktop app for synchronized music playback.

## Task: Project Scaffolding

Initialize the full project structure for a Tauri 2 + Svelte application with a Rust backend.

### What to do:

1. **Initialize a Tauri 2 project** with Svelte as the frontend framework. Use the Tauri CLI or set it up manually:
   - `src/` directory for the Svelte frontend
   - `src-tauri/` directory for the Rust backend

2. **Set up the Svelte frontend:**
   - `package.json` with Svelte, Vite, and TypeScript dependencies
   - `svelte.config.js`
   - `vite.config.ts`
   - `tsconfig.json`
   - `src/main.ts` — Svelte entry point
   - `src/App.svelte` — Root component (just a placeholder "PhaseLock" heading for now)

     ```toml
     bincode = "1"
     mdns-sd = "0.11"

     log = "0.4"
     parking_lot = "0.12"
     uuid = { version = "1", features = ["v4", "serde"] }
     sha2 = "0.10"
     ```
   - `tauri.conf.json` with:
     - App name: "PhaseLock"
     - Bundle identifier: "com.phaselock.app"
     - Version: "0.1.0"
     - Window title: "PhaseLock"
     - Default window size: 1024x768
